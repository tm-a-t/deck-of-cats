# Пираты — Deck Builder

Веб-игра на Phaser 3, пиксель-арт. `index.html` + скрипты в `js/`.

## Правила

- Жанр: deck-building, но вместо карт — пираты (члены команды).
- **Раунд**: корабль с 5 пиратами (из колоды) прибывает к случайному острову.
  1. **Высадка**: игрок отправляет 0–2 пиратов на остров (по одному, клик = отправка). Действие мгновенное.
  2. **Корабль**: оставшиеся пираты выполняют корабельное действие одновременно.
  3. **Магазин**: игрок покупает новых пиратов за энтузиазм (☠️). Энтузиазм сбрасывается, ресурсы остаются.

## Стартовая колода (10 пиратов)

| Тип | Кол-во | Остров | Корабль |
|-----|--------|--------|---------|
| Дровосек | 3 | 90% → 1🪵 | 3🪵 → 3🗡️ |
| Шахтёр | 3 | 90% → 1🪨 | 3🪨 → 1💣 |
| Искатель | 2 | → 1☠️ | 2🪵+2🪨 → 3☠️ |
| Юнга | 2 | не может | → 1☠️ |

## Острова (случайный каждый раунд)

- Лесной — x2 к добыче 🪵
- Скалистый — x2 к добыче 🪨
- Остров сокровищ — x2 к добыче 🪙
- Портовый — можно отправить 3 пиратов вместо 2
- Черепов — +2☠️ энтузиазма в начале раунда

## Абордаж (каждый 5-й раунд)

Каждый 5-й раунд (5, 10, 15…) вместо острова появляется вражеский корабль.
- У каждого пирата есть **сила** (⚔️), отображается везде под эффектами.
- Все 5 пиратов в руке сражаются: суммарная сила команды должна быть **не меньше** силы вражеского корабля.
- Сила вражеского корабля = `5×N`, где N — номер абордажа (5, 10, 15, 20…).
- **Победа** → раунд заканчивается, начинается следующий (без магазина).
- **Поражение** → Game Over.

| Тип | ⚔️ | ☠️ цена | Корабль |
|-----|-----|---------|---------|
| Дровосек | 1 | — | 3🪵 → 3🗡️ |
| Шахтёр | 1 | — | 3🪨 → 1💣 |
| Искатель | 1 | — | 2🪵+2🪨 → 3☠️ |
| Юнга | 0 | — | → 1☠️ |
| Плотник | 1 | 3 | 1🪵 → 3🗡️+3☠️ |
| Каменщик | 1 | 3 | 1🪨 → 💣+3☠️ |
| Боцман | 1 | 5 | → 3☠️ |
| Квартирмейстер | 4 | 10 | → 2☠️ |
| Лесоруб | 2 | 7 | 2🪵 → 6🗡️+4☠️ |
| Старатель | 2 | 7 | 2🪨 → 2💣+4☠️ |
| Контрабандист | 2 | 8 | 1🪙 → 5☠️ |
| Исследователь | 1 | 9 | 1🪙 → 6☠️ |
| Мастер-дровосек | 3 | 13 | 2🪵 → 9🗡️+4☠️ |
| Мастер-шахтёр | 3 | 13 | 2🪨 → 3💣+4☠️ |
| Охотник за сокровищами | 4 | 17 | 1🪙 → 7☠️ |
| Головорез | 3 | 5 | 2🪙 → изгнать 1 пирата |

**Квартирмейстер** — остров: возвращает 1 пирата с острова обратно на корабль (отзыв). Если никого нет — эффекта нет.

**Головорез** — остров: → 1☠️. Корабль: тратит 2🪙, игрок выбирает пирата из общей команды (не из руки) и навсегда удаляет его из колоды. Если нет 2🪙 или нет кандидатов — эффекта нет.

## Вооружение корабля

Когда пират тратит ресурс на корабле (корабельное действие), помимо энтузиазма производится вооружение:

- **Оружие** (🗡️): производится при трате 🪵. Даёт **+1⚔️** в ближайшем абордаже. После боя все оружие расходуется.
- **Пушка** (💣): производится при трате 🪨. Даёт **+1⚔️** во всех будущих абордажах (навсегда).

Бонус вооружения суммируется с силой пиратов при абордаже. Отображается рядом с инвентарём.

## Промах на острове

Если пират не добыл целевой ресурс → получает 1 случайного другого ресурса (или карту сокровищ 🗺️ с шансом 1%). Карта автоматически тратится при отправке искателя — даёт +30% к шансу золота.

## Архитектура

- Разрешение: 960×1440 (Scale.FIT)
- Глобальное состояние в объекте `G`
- Одна Phaser-сцена `GameScene`
- UI перерисовывается целиком через `renderAll()` при каждом изменении стейта
- Контейнеры: `top`, `island`, `phase`, `hand`, `btn`, `shop`, `tip`, `fx`, `gameover`

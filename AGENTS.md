# Пираты — Deck Builder

Веб-игра на Phaser 3, пиксель-арт. `index.html` + скрипты в `js/`.

## Где смотреть правила игры

Все геймплейные правила вынесены в [docs/game-rules.md](docs/game-rules.md). Это источник истины по правилам.

- Базовый цикл, маршрут, фазы раунда: раздел `Основной геймплей`.
- Стартовая колода: раздел `Стартовая колода`.
- Типы островов: раздел `Острова`.
- Абордаж и сила вражеского корабля: раздел `Абордаж`.
- Характеристики, цены и эффекты пиратов: таблица и примечания в разделе `Абордаж`.
- Вооружение корабля (🗡️/💣): раздел `Вооружение корабля`.
- Промах на острове и карта сокровищ: раздел `Промах на острове`.
- Ограничения спрайтшита и `DEFAULT_FRAME`: раздел `Спрайтшит`.

## Обязательные инструкции агенту

- Если есть любые вопросы по игровым правилам, сначала смотри [docs/game-rules.md](docs/game-rules.md).
- Если вносишь любое изменение геймплея (новый пират, изменение цены/силы/эффекта, ресурсы, острова, фазы раунда, абордаж, вооружение, шансы и т.д.), **ВСЕГДА** обновляй [docs/game-rules.md](docs/game-rules.md) в этом же изменении.
- Изменение геймплея считается неполным, пока соответствующее правило не зафиксировано в [docs/game-rules.md](docs/game-rules.md).

## Архитектура

- Разрешение: 960×1440 (Scale.FIT)
- Глобальное состояние в объекте `G`
- Базовая сцена `GameScene` + модальные сцены `MapScene` и `ShopScene`
- UI перерисовывается целиком через `renderAll()` при каждом изменении стейта
- Контейнеры `GameScene`: `top`, `island`, `phase`, `hand`, `btn`, `nav`, `tip`, `fx`, `gameover`
- Основной layout:
  - верх: статус, инвентарь, вся команда;
  - центр: текущий остров/вражеский корабль/открытое море;
  - низ: рука пиратов;
  - самый низ: кнопки `Map` и `Shop`;
  - status/help-текст всегда над рукой.
- Модалки (`Map`, `Shop`) закрываются только по клику вне "листа бумаги" или кнопкой `✕`.

## Файлы (js/)

| Файл | Содержимое |
|------|-----------|
| `js/constants.js` | `BG_COLOR`, `RES_EMOJI`, `ISLANDS`, `TYPES`, `SHOP_POOL` |
| `js/map.js` | Генерация и утилиты карты маршрута (`generateMap`, `getAvailableNodes`, ...) |
| `js/state.js` | `G`, `mkP`, `initState`, `randomShopType`, `initialShop` |
| `js/layout.js` | `REF_H`, `computeLayout` |
| `js/scene.js` | `GameScene` — основной игровой цикл и базовый UI |
| `js/mapScene.js` | `MapScene` — модалка карты (выбор узла или просмотр маршрута) |
| `js/shopScene.js` | `ShopScene` — модалка магазина и анимации витрины |
| `js/main.js` | Инициализация `Phaser.Game` |

# Пираты — Правила игры

Этот документ — источник истины по геймплею.

Если в коде изменён геймплей (новый пират, цены, эффекты, баланс, фазы, ресурсы, острова, абордажи и т.д.), это изменение **обязательно** нужно отразить в этом файле в том же изменении.

## Основной геймплей

- Жанр: deck-building, но вместо карт — пираты (члены команды).
- **Маршрут**: всегда линейный, без развилок и без выбора узла игроком.
  1. В конце раунда игра переходит в служебную фазу `map` и сразу автоматически выбирает следующий узел.
  2. Кнопки `Map` в интерфейсе нет, модалка карты для игрока не используется.
  3. Каждый 5-й узел маршрута — вражеский корабль, остальные — острова.
- **Раунд**: корабль с 5 пиратами (из колоды) прибывает к острову или вражескому кораблю.
  1. **Высадка**: игрок отправляет 0–2 пиратов на остров (по одному, клик = отправка).
     У отправленных на остров срабатывает верхний эффект пирата (островной).
  2. **Корабль**: оставшиеся пираты выполняют корабельные действия по очереди.
     У оставшихся на корабле срабатывает нижний эффект пирата (корабельный).
  3. **Магазин**: игрок открывает модалку магазина кнопкой `Shop`, покупает пиратов за энтузиазм (☠️), затем нажимает `Next round` на основном экране.
     При покупке витрина магазина анимируется: купленный пират исчезает, остальные сдвигаются, новый заезжает справа.
  4. После `Next round` энтузиазм сбрасывается, ресурсы сохраняются.
- **Пауза**:
  1. Кнопка `Menu` находится в правом верхнем углу.
  2. При нажатии открывается пауза-меню с кнопками `Continue` и `Restart`.
  3. `Continue` закрывает паузу и отправляет `PokiBridge.gameplayStart()`.
  4. `Restart` перезапускает текущий режим (обучение или обычный ран) с начала.
  5. Открытие паузы отправляет `PokiBridge.gameplayStop()`.
- Режим после обучения: **бесконечная игра** (финального экрана победы по количеству пройденных кораблей нет).

## Обучение (первые 5 ходов)

- Игра при запуске всегда стартует сразу с обучения (главного меню перед стартом нет).
- Первые 5 ходов — **scripted tutorial**: шаги и контент фиксированы, без рандома.
- Объясняющие тексты обучения показываются через два формата:
  1. **Блокирующие popup-слайды** (оверлей) только для критичных стартовых инструкций.
  2. **Неблокирующие подсказки** (informer-card) для большей части обучения.
- На 1-м ходу после отправки **первого** пирата показывается неблокирующая подсказка, что полученные ресурсы добавляются в верхнюю панель ресурсов.
- На 1-м ходу переход из высадки в фазу корабля выполняется через первый ручной `End landing`; после него показывается неблокирующая подсказка, что у пиратов на корабле срабатывает нижний эффект.
- Обычная фазовая строка игры не заменяется tutorial-текстами.
- В первых ходах обязательно показываются острова с бонусом **x2** к добыче, чтобы обучить модификаторам острова.
- До момента mismatch в обучении заранее зафиксированы ожидаемые исходы.
- На 3-м ходу обучение ведёт к покупке **featured pirate** (ключевая покупка туториала).
- Featured-пират обучения: **Admiral Blackpowder** (`5☠️`, корабль: `1🪙 -> +3💣`).
- На 3-м ходу в магазине показывается только один пират (featured), а его эффекты прокручиваются текстом по очереди.
- В тексте обучения акцент делается на общей концепции: покупка пиратов усиливает колоду и будущие ходы.
- На 4-м ходу обучение требует розыгрыша купленного пирата.
- На 4-м ходу также фиксируется обязательный `mismatch`: ожидали 🪵, но выпало 🪙.
- До `mismatch` игрок не получает предваряющих подсказок о «не том» ресурсе; объяснение даётся после факта `mismatch`.
- Важно: mismatch не объясняется «особым островом»; лор обучения прямо говорит, что такой исход возможен на **любом** острове.
- На 5-м ходу появляется первый босс (первый обучающий абордаж).
- После победы в обучающем абордаже показывается плашка `Tutorial Complete`; обычный бесконечный ран стартует только после нажатия `Continue`.
- После завершения обучения допустим небольшой остаток ресурсов: это нормальное состояние, не ошибка.

## Стартовая колода (10 пиратов)

| Тип | Кол-во | Остров | Корабль |
|-----|--------|--------|---------|
| Дровосек | 4 | 90% → 1🪵 | 3🪵 → 2☠️ |
| Шахтёр | 4 | 90% → 1🪨 | 3🪨 → 1💣 |
| Юнга | 2 | не может | → 1☠️ |

## Острова (случайный каждый раунд)

- Лесной — x2 к добыче 🪵
- Скалистый — x2 к добыче 🪨
- Остров сокровищ — x2 к добыче 🪙
- Портовый — можно отправить 3 пиратов вместо 2
- Черепов — +2☠️ энтузиазма в начале раунда

## Абордаж (каждый 5-й раунд)

Каждый 5-й раунд (5, 10, 15…) вместо острова появляется вражеский корабль.
- У каждого пирата есть **сила** (⚔️), отображается везде под эффектами.
- Все 5 пиратов в руке сражаются: суммарная сила команды должна быть **не меньше** силы вражеского корабля.
- Сила вражеского корабля = `trunc(4 × N^1.2 + 2)`, где N — номер абордажа (1, 2, 3, ...).
- **Победа** → раунд заканчивается, начинается следующий (без магазина).
- **Поражение** → Game Over.

**Старьёвщик** — остров: → 1🗡️ (гарантировано). Корабль: тратит 1💣, производит 3🪨+2☠️.

**Квартирмейстер** — остров: возвращает 1 пирата с острова обратно на корабль (отзыв). Если никого нет — эффекта нет.

**Головорез** — остров: → 1☠️. Корабль: тратит 2🪙, игрок выбирает пирата из общей команды (не из руки) и навсегда удаляет его из колоды. Если нет 2🪙 или нет кандидатов — эффекта нет.

**Рейдер** — остров: → 3🗡️ (гарантировано). Корабль: теряется навсегда (удаляется из колоды).

**Барышник** — остров: 1🪙 → 2🪙 (конвертация). Корабль: теряется навсегда.

**Скиталец** — остров: → 2🪵 (гарантировано). Корабль: теряется навсегда.

**Высадчик** — остров: изгоняет предыдущего пирата на острове (навсегда удаляет из колоды). Если никого нет — эффекта нет. Корабль: → 3🗡️.

**Дозорный** — остров: тянет 1 пирата из колоды в руку. Корабль: 1🪵 → 3🗡️.

**Выживальщик** — остров: 1🪵 (90% шанс) + гарантированно 2☠️. Корабль: → 2☠️.

## Вооружение корабля

Когда пират тратит ресурс на корабле (корабельное действие), помимо энтузиазма производится вооружение:

- **Оружие** (🗡️): производится при трате 🪵. Даёт **+1⚔️** в ближайшем абордаже. После боя всё оружие расходуется.
- **Пушка** (💣): производится при трате 🪨. Даёт **+1⚔️** во всех будущих абордажах (навсегда).

Бонус вооружения суммируется с силой пиратов при абордаже. Отображается рядом с инвентарём.

## Промах на острове

Если пират не добыл целевой ресурс → получает 1 случайного другого ресурса (или карту сокровищ 🗺️ с шансом 1%). Карта автоматически тратится при отправке искателя — даёт +30% к шансу золота.
